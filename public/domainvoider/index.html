DomainVoider

DomainVoider is dedicated to providing AdGuard & AdGuard Pro for iOS users with a custom, privacy & security focused, simplified ad filter. DomainVoider combines a modified version of AdGuard's Simplified Domain Names filter with a list of active phishing domains licensed from Hardenize.com

DomainVoider is also for use with uBlock Origin or AdGuard for Firefox. When used with either extension, the included $CSP rules frame-ancestors 'none', upgrade-insecure-requests & block-all-mixed-content help to increase web security without breaking websites. A Blocked by Content Security Policy notice will be seen on websites that practice poor security measures. Our four main Content Security Policy rules:

  • frame-ancestors 'none' protects against clickjacking attacks,
  • upgrade-insecure-requests upgrades capable websites from http:// to https:// connections,
  • block-all-mixed-content protects against resources loaded over http:// into sites that use https://,
  • object-src 'none' prevents plugins such as Adobe Flash - an ongoing source of security holes - that attempt to load using object, embed & applet tags.

This will not interfere with browser extensions such as AdGuard, uBlock Origin, Decentraleyes, ClearURLs & HTTPS Everywhere. It only restricts plugins, none of which are necessary for any modern website to function.
