Malicious URLs, Domains & Hosts Blocklists Mirror

This is a mirror of Urlhaus Filters.

The Main Repo & this Mirror's Repo are hosted using GitLab. This mirror is built, cached & served using Cloudflare Workers. Each filter list is cached & served by Fastly via Statically.io.

The three main variations available are:

  • Domains (plain);
  • URLs (adblock syntax);
  • Hosts (0.0.0.0)

Each of the above three formats is available in an online-only or combination online/offline list. The recommended format is the larger, online/offline format whenever possible.

All subscriptoion links are now fully compatible witb AdGuard iOS & AdGuard Pro iOS.
