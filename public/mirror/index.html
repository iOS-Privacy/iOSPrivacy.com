Malicious Domains & Hosts Mirror

This is a mirror of Urlhaus Filters.

The main repo & this mirror's repo are hosted using GitLab. This mirror is built, cached & served using Cloudflare Workers.

The three main variations available are:

  • Domains (plain);
  • URLs (adblock syntax);
  • Hosts (0.0.0.0)

Each of the above three formats is available in an online-only or combination online/offline list. The recommended format is the larger, online/offline format whenever possible.
