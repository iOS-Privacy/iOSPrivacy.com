Malicious URLs, Domains & Hosts Blocklists Mirror

This is a mirror of Urlhaus Filters.

The Main Repo & this Mirror's Repo are hosted using GitLab. This mirror is built, cached & served using Cloudflare Workers. Each filter list is cached & served by Fastly via Statically.io.

The variations available are:

  • Domains
  • URLs
  • Hosts
  • Bind
  • dnsmasq
  • Unbound

Each of the above formats is available in an online-only or combination online/offline list. The recommended format is the larger, online/offline format whenever possible.

All subscription links are fully compatible with AdGuard iOS & AdGuard iOS Pro. Clicking one will open a dialogue just like desktop browser adblocking extensions.
