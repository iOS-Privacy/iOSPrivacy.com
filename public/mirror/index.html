Urlhaus Filters Mirror

This mirror hosts lists built from data available from Urlhaus.

The Main Repo & This Mirror's Repo are hosted using GitLab. This mirror and the lists within are built, cached & served using Cloudflare Workers. Each list is cached & ultimately served using our own CDN.

The variations available are:

  • Domains
  • URLs
  • Hosts

Each of the above formats is available in an online-only or combination online/offline list. The recommended format is the larger, online/offline format whenever possible.

All subscription links are fully compatible with AdGuard iOS & AdGuard iOS Pro. Clicking one will open a dialogue just like desktop browser adblocking extensions.

This mirror serves only those lists compatible with browser addons such as uBlock Origin & AdGuard. See https://gitlab.com/iosprivacy/urlhaus-filter-mirror for Bind, dnsmasq, Snort, Suricata, & other formats.
