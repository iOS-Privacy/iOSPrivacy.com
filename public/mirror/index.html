Malicious URLs, Domains & Hosts Blocklists Mirror

This is a mirror of Urlhaus Filters.

The Main Repo & this Mirror's Repo are hosted using GitLab. This mirror is built, cached & served using Cloudflare Workers. Each filter list is cached & served by Fastly via Statically.io.

The three main variations available are:

  • Domains (plain);
  • URLs:
    There are three URL Blocklists. One uses the $all modifier, another uses the $document modifer which & the third uses a simplified format.
    Each can be used with AdGuard Home, AdGuard & uBlock Origin.
    For AdGuard Pro & AdGuard iOS, the online simplified version is good as a Safari content blocker & the complete simplified version works well on the DNS level for AdGuard Pro & AdGuard iOS;
  • Hosts (0.0.0.0)

Each of the above formats is available in an online-only or combination online/offline list. The recommended format is the larger, online/offline format whenever possible.

All subscription links are now fully compatible with AdGuard iOS & AdGuard Pro iOS.
